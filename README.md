# 📚 AI Powered Study Buddy

An AI-powered study assistant that helps students with notes, questions, summaries, and learning support using artificial intelligence.

---

## 🚀 Features

- 🤖 AI chatbot for answering questions
- 📝 Notes summarization
- 📖 Concept explanation
- 🎯 Exam preparation support
- 💻 Simple and user-friendly interface

---

## 🛠 Technologies Used

- Python
- OpenAI API
- Streamlit / Flask
- HTML, CSS

---

## 📂 Project Files

- app.py → Main application
- README.md → Project documentation
- requirements.txt → Required libraries

---

## ▶ How to Run

Step 1: Install libraries

pip install -r requirements.txt

Step 2: Run program

python app.py
